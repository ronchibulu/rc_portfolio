| v3 (old) | v4 (new) | Affected components |
|----------|----------|---------------------|

| `outline-none` | `outline-hidden` | Button focus state |
| `ring` (bare) | `ring-3` | All interactive focus rings |
| `bg-[--var]` | `bg-(--var)` | Any CSS-variable class |
| `text-[--var]` | `text-(--var)` | Any CSS-variable class |
| `border-[--var]` | `border-(--var)` | Any CSS-variable class |

**Grep patterns for the audit step (exact regex, word-boundary-sensitive):**
- `className=.*\b(shadow|rounded|ring)\b(?![-\w])` — catches bare class names needing migration
- `outline-none` — direct string
- `\[--[a-z-]+\]` — old CSS-var syntax `[--foo]` (replace surrounding `[ ]` with `( )`)

**Shadcn `cn` helper contract (already created in Plan 02):**
```ts
// src/lib/utils.ts
```
