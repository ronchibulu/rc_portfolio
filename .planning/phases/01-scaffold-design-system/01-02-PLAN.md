    /* Dark-only baseline. Set on <html> by BaseLayout.astro class="dark" AND reinforced here
     * so even if the class is missing, the site still shows zinc-950. (D-22)

      background-color: var(--color-zinc-950);
      color: var(--color-zinc-100);
      font-family: var(--font-sans); /* Tailwind v4 default system stack — font-pixel is opt-in */
